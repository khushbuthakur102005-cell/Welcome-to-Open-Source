# Welcome to Open Source 🚀

Hello Developers!

This repository is created for beginners who want to learn Open Source Contributions using Git and GitHub.

## 📌 What is Open Source?

Open Source is software whose source code is publicly available and anyone can contribute to it.

## 🚀 How to Contribute

1. Fork this repository
2. Clone your fork
3. Make changes
4. Commit changes
5. Push changes
6. Create a Pull Request

## 🛠 Technologies

- Git
- GitHub
- Open Source
